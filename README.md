This is a light weight Prototype Attendance System for Lecturers in Univeristies to take Attendance via geolocations.
It has IP and device key tracking and other security measures to be able to detect devices and stop a device from taking the attendance twice.
It uses google sheets via google script for the DB.
Upscale and commercial Build is on going and would be available to the public soon in the name - Locify - Pressence made Simple.
